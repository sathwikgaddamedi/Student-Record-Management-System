# Student Record Management System

## Project Documentation

This project encompasses a system for managing student records efficiently. It includes features such as adding, updating, retrieving, and deleting student information, along with functionality for sorting and searching records.

## Features
- Add new student records
- Update existing records
- Delete student records
- Search and sort records

## Technologies Used
- Language: Python
- Database: SQLite
- Framework: Flask

## Getting Started
1. Clone the repository using `git clone https://github.com/sathwikgaddamedi/Student-Record-Management-System.git`
2. Install the required dependencies using `pip install -r requirements.txt`
3. Run the application using `python app.py`
